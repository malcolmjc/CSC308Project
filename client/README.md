# Forest Run Game Repository

## Structure

* [client](client)
* [server](server)

## Installation

```bash
// Install Client Package
cd client && npm install

// Install Server Package
cd server && npm install
```

## Run Client

```bash
cd client
npm run build

// Use build:prod for production
```

## Run Server

```bash
cd server
npm run start:dev

// Use start for production
```
